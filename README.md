# WX_ReactNative
模仿微信的React-native项目
## Start Project
>项目启动

```

第一步：

git clone https://github.com/liwudi/WX_ReactNative.git

第二步：

cd WX_ReactNative

第三步：

npm install

第四步：

react-native run-ios || react-native run-android

```
